# Clina-App

## Project setup
```
Apos Baixar o projeto ou clona-lo entre na pasta crud onde voce ira se deparar com outras duas pastas e em seu terminal(cmd ou outro de sua escolha) insira o comando em ambas as pastas  -> npm install

p.s lembrando que é necessario ter instalando o node.js para utilizar os comando npm. 
```

### Compiles and hot-reloads for development
```
Para Iniciar a Aplicação na pasta api mockada abra o terminal(cmd ou outro de sua escolha) e insira o comando-> npm run start-auth (nesse terminal você pode acompanhar as requisições get e post).

então vá a pasta frontend abra o terminal e abra o terminal(cmd ou outro de sua escolha) e insira o comando --> npm run serve
A aplicação sera compilada e iniciada, então no terminal ira aparecer esta mensagem : " App running at:
                                                                                        - Local:   http://localhost:8080/
                                                                                        - Network: http://192.168.0.19:8080/"
Copie a sentença que esta na frente de "local:" e cole em seu navegador, o site sera carregado e tudo vai estar pronto para que você siga o fluxo. 
p.s: as portas do localhost podem mudar de acordo com a disponibilidade por isso sugiro copiar e colar ao inves de digitar. 
                                                                                        
```
